# Weather App 🌦️

A simple weather app made using HTML, CSS and JavaScript.  
It shows live weather details of any city using the OpenWeather API.   

## Features
- Search weather by city name
- Temperature, humidity and wind speed
- Weather icons and dynamic backgrounds
- Responsive design for mobile and desktop
- Error message for invalid cities

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## Project Files
- `index.html` → structure
- `styles.css` → design
- `script.js` → weather functionality

Made as a practice project for learning frontend development.
